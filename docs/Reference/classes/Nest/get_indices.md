# Accera v1.2.13 Reference

## `accera.Nest.get_indices()`
Gets the iteration space dimensions for a nest.

## Returns
Tuple of `Index`

## Examples

Get the iteration space dimensions for a 3-dimensional nest:

```python
i, j, k = nest.get_indices()
```
